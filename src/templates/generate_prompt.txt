import requests

    session.timeout = {timeout}
    return session

## 代码规范

1. **用例ID标记**: 每个测试函数上方必须有 `# TestCase: TC-XXX` 注释
2. **函数命名**: `test_{{场景英文描述}}`
3. **类组织**: 同一API的测试放在同一个类中
4. **断言明确**: 使用具体的断言条件，避免模糊判断
5. **数据独立**: 每个测试函数应独立，不依赖执行顺序
